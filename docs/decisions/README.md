# Architecture Decisions

This directory records lightweight ADRs (Architecture Decision Records) for major design choices.

Naming: `YYYY-MM-DD-short-title.md`

Template:

```md
# Title

Date: YYYY-MM-DD

## Context
- What problem are we solving?
- What constraints matter?

## Decision
- What did we choose and why?

## Alternatives Considered
- Option A: pros/cons
- Option B: pros/cons

## Consequences
- Expected benefits
- Trade-offs / risks
```
